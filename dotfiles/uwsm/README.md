# UWSM

[UWSM](https://github.com/Vladimir-csp/uwsm) (Universal Wayland Session Manager)
configuration for starting Hyprland as a systemd service.

## Target

`~/.config/uwsm/`

## Install

```bash
stow --target="$HOME" --dir=dotfiles uwsm
```

## Notes

The `default` file specifies which compositor UWSM starts (Hyprland).
