# Waybar

[Waybar](https://github.com/Alexays/Waybar) status bar configuration.

## Target

`~/.config/waybar/`

## Install

```bash
stow --target="$HOME" --dir=dotfiles waybar
```

## Key Files

| File | Description |
|---|---|
| `config.jsonc` | Bar layout, modules, and behavior |
| `style.css` | Bar appearance and styling |

## Dependencies

- [Waybar](https://github.com/Alexays/Waybar)
